# StoryboardLab

## rbenv

```sh
$ brew install rbenv
$ rbenv init

# Close your Terminal window and open a new one.

$ rbenv install 3.4.5
$ rbenv local 3.4.5
$ gem install bundler
```

## Init bundle

```sh
$ bundle init
```

## Add gem to the Gemfile and run bundle install

```sh
$ bundle add cocoapods
```
